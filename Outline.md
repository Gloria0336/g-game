## 三、AI 提示詞風格規範

### 3.1 通用約束（所有 AI 段落皆適用）

1. 當前 DES 值對應語氣模式（見 3.3）
2. 禁止描述角色死亡（結局場景除外）
3. 禁止添加場景骨架中不存在的行動
4. 禁止矛盾當前戰鬥結果（勝 / 負 / 逃脫）
5. 序章期間：女主角無名字、無過去記憶可引用
6. 惡魔對話必須符合角色聲音規範（見 3.2）

### 3.2 角色聲音規範

#### 旁白（narrator）
- 視角：第三人稱限知，貼近女主角心理，統一稱「妳」
- 句式：短句優先，動詞精確，少用形容詞堆疊
- 禁止：心靈雞湯式總結、過度詩意化、直接陳述情感

#### 瑠夜（demon_a）
- 特徵：輕柔帶笑，每句話都像精心設計的陷阱，越親近越危險
- 句式：語尾常帶停頓或反問，溫柔的話語裡永遠藏著邏輯陷阱
- 禁止：直接表達情緒、粗話、語氣急促

#### 颯牙（demon_b）
- 特徵：直接強硬，把「比你強」當口頭禪，情感表達笨拙
- 句式：短促有力，常用否定句，偶爾說了軟話立刻自打臉
- 禁止：溫柔說話方式、長篇大論、細膩的情感描述

#### 玄冥（demon_c）
- 特徵：話極少，行動比話快，沉默背後有說不出口的事
- 句式：最少字數傳達最多信息，沉默也是一種回答
- 禁止：多餘解釋、主動說明情感、長句

### 3.3 DES 語境語氣模板

| DES範圍 | 語氣模式 | 提示詞關鍵指令 |
|---------|---------|--------------|
| 0–40 | 清醒 | 語氣冷靜客觀，旁白不帶情感傾向，惡魔互動克制 |
| 41–80 | 動搖 | 旁白帶輕微疲憊與矛盾感，惡魔開始過度注意女主角狀態 |
| 81–120 | 心防崩解 | 旁白帶情感暗示，惡魔行為越界感增強，描寫肢體距離細節 |
| 121–160 | 依賴萌生 | 情感流露明顯，女主角開始主動靠近，帶肢體細節描寫 |
| 161–200 | 完全沉淪 | 18+ 語境，須設定允許方可觸發 |

```
任務：戰鬥結束後的旁白敘事，60–150 字。
戰鬥結果：{win / lose}
DES 值：{n}（語氣模式：{mode}）
當前 HP 比例：{n}%
約束：[通用約束 1–5]
```

#### 惡魔停留對話（demon_dialogue）
```
任務：生成惡魔 {demonId}（{demon_name}）戰後停留時的一段對話。
     需包含惡魔台詞 2–4 句，及 3 個玩家可選的回應方向（每項 10–20 字）。
惡魔設定：{CHAR_PROFILES[demonId]}
當前數值：heroine_axis={n} / demon_axis={n} / trust={n}
DES 值：{n}（語氣模式：{mode}）
contract_status：{status}
約束：[通用約束 1–6] + 角色聲音規範
```

#### 場景填充（scene_fill）
```
任務：為以下場景骨架填充旁白與對話，字數 {min}–{max} 字。
場景 ID：{sceneId} ／ 背景：{background}
當前數值：heroine_axis={n} / demon_axis={n}
DES 值：{n}（語氣模式：{mode}）
約束：[通用約束 1–6]

骨架：
{sceneData.dialogues（僅含 speaker 與空白 text 的版本）}
```

#### 結局敘事（ending_narrative）
```
任務：為結局「{ending_name}」撰寫最終敘事，200–400 字。
路線：{route}（{demon_name} 路線 / 獨行路線）
結局軌道：{ending_track}（傾心HE / 惡魔傾心 / 敵對NE / 背叛BE）
最終數值快照：heroine_axis={n} / demon_axis={n} / trust={n} / DES={n}
約束：[通用約束 1–4]
      - 允許描述角色命運（結局場景）
      - 語氣呼應整段路線的情感弧線
```
